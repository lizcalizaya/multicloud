Aplicación Web MultiCloud

Esta aplicación demuestra una arquitectura MultiCloud con AWS, OCI y Azure.

  • AWS EC2/VPC: alojamiento de la aplicación web.
  • OCI Autonomous Database: almacenamiento de datos estructurados.
  • Azure Blob Storage: almacenamiento de archivos o multimedia.
  • GitHub Actions: despliegue automatizado CI/CD.
  • OWASP: controles de seguridad en la capa de aplicación.
